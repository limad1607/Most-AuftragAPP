# Most Außendienst App

Vite + React + TypeScript Projekt für deine Außendienst-Webapp.

## Start lokal

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Enthalten

- Auftragserfassung
- Kundenstamm
- Artikelstamm
- Tagesbericht
- Archiv mit Monats- und Tagesgruppen
- Backup Export / Import
- PDF-Export mit jsPDF
- Offline-Speicherung via IndexedDB

## Hinweis

Dieses Projekt ist bewusst schlank gehalten und nutzt keine zusätzlichen UI-Frameworks wie shadcn/ui oder Tailwind. Dadurch ist es einfacher auf GitHub zu pflegen und lokal zu starten.
